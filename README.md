# FuelEU Maritime Compliance (WIP)

## Overview

This project is an implementation of a FuelEU Maritime compliance system.

The goal is to build:

* Backend APIs for routes, compliance balance, banking and pooling
* Frontend dashboard for visualization

Currently focusing on backend first.

## Tech Stack

* Node.js + TypeScript
* PostgreSQL (planned)
* React (frontend later)
